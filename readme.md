# KittyBot — Telegram бот с котиками и не только

Простой Telegram-бот, который показывает текущее время, отправляет случайные изображения котиков и генерирует случайные цифры.  
Бот написан на Python с использованием библиотеки `pyTelegramBotAPI` (telebot).


## 🚀 Пошаговая инструкция по установке и запуску

### 1. Создание бота в Telegram (через BotFather)

1. Откройте Telegram и найдите пользователя [@BotFather](https://t.me/botfather).
2. Отправьте команду `/newbot`.
3. BotFather попросит ввести имя вашего бота (например, `MyKittyBot`). Это имя будет отображаться в списке контактов.
4. Затем введите **username** бота — он должен заканчиваться на `bot` (например, `MyKittyBot_bot`). Username используется для ссылок и поиска.
5. После успешного создания BotFather пришлёт сообщение с **токеном** — длинной строкой вида `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`. **Сохраните этот токен**, он понадобится на следующем шаге.

> 🔗 [Документация BotFather](https://core.telegram.org/bots#6-botfather)

### 2. Получение токена и создание файла `.env`

Токен — это ключ доступа к вашему боту. Хранить его прямо в коде небезопасно, поэтому мы используем переменные окружения.

1. В корневой папке проекта создайте файл с именем `.env`.
2. Откройте этот файл в текстовом редакторе и добавьте следующую строку:
   ```
   TOKEN=ваш_токен_здесь
   ```
   Замените `ваш_токен_здесь` на токен, полученный от BotFather. Например:
   ```
   TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
   ```
3. Сохраните файл.

> 🔗 Библиотека для работы с `.env` — [python-dotenv](https://pypi.org/project/python-dotenv/) (она уже включена в зависимости).
